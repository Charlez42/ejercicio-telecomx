# Informe de análisis de churn de clientes - Telecom X

## ¿Qué hace este proyecto?

Este repositorio contiene un análisis completo y estructurado del fenómeno de **abandono de clientes (churn)** en Telecom X.

El trabajo se centra en:

- **Entender** por qué los clientes deciden cancelar el servicio
- **Identificar** los principales factores y patrones de comportamiento asociados a la fuga
- **Detectar** momentos y perfiles de mayor riesgo de abandono
- **Proponer** acciones concretas de retención basadas en evidencia

### Etapas principales del informe

1. **Preparación y limpieza de datos**  
   Transformación de estructura anidada (JSON/diccionarios), tratamiento de valores ausentes, estandarización de formatos y corrección de tipos de datos para garantizar calidad analítica.

2. **Análisis exploratorio (EDA)**  
   Exploración visual y estadística univariada, bivariada y multivariada para descubrir relaciones entre las características de los clientes y la variable de churn.

3. **Extracción de insights accionables**  
   Identificación de segmentos vulnerables, drivers clave de abandono y barreras protectoras que favorecen la permanencia.

4. **Recomendaciones estratégicas**  
   Propuestas concretas de intervención comercial, operativa y de experiencia del cliente para reducir la tasa de churn de forma proactiva.

## Finalidad del proyecto

El objetivo central es **convertir datos en decisiones de retención** que permitan a Telecom X:

- Anticiparse a la fuga en lugar de reaccionarla
- Priorizar esfuerzos y recursos en los clientes y momentos de mayor riesgo
- Diseñar intervenciones dirigidas (onboarding, pricing, incentivos contractuales, métodos de pago, etc.)
- Sentar las bases para futuros modelos predictivos de churn (machine learning)
